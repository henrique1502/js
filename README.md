# Projeto IF, ELSE, E e OU

Este projeto foi desenvolvido utilizando HTML e JavaScript para demonstrar o uso de estruturas condicionais.

## Conteúdos utilizados

- if
- else
- Operador lógico E (&&)
- Operador lógico OU (||)

## Funcionamento

O usuário informa:

- Sua idade
- Seu time de futebol

O sistema verifica:

1. Se é maior ou menor de idade.
2. Se torce para o Palmeiras.
3. Se torce para Palmeiras ou Flamengo.
4. Se é maior de idade e torce para o Palmeiras.

## Arquivos do projeto

- `index.html` → Estrutura da página.
- `main.js` → Código JavaScript com as verificações.
- `README.md` → Documentação do projeto.

## Como executar

1. Abra o arquivo `index.html` em um navegador.
2. Digite a idade e o time.
3. Clique no botão **Verificar**.
4. Observe os resultados exibidos na tela.
